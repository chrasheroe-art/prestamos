🔔 Alertas automáticas por WhatsApp
Para recibir alertas en tu WhatsApp necesitas activar un trigger diario en Apps Script y registrarte gratis en callmebot.com para obtener tu API key.
Pasos para configurar
  1. Abre tu Google Sheet → Extensiones → Apps Script
  2. Borra el código existente y pega el que copias arriba
  3. Implementar → Nueva implementación → Aplicación web
  4. Acceso: Cualquiera → Implementar → copia la URL
  5. Para alertas: Triggers (ícono reloj) → Agregar trigger → checkAlerts → Diariamente
